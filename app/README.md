# Imaging Directory App
This directory contains the source code for an imaging directory application, which will be used to location imaging facilities for a patient.

# Stack

## Databse
Postgresql with postgis

## Backend
Python + Django

## API
Django rest framework

## Frontend
Typescript + Next.js (React)

## UI
Shadcn

# Infrastructure
- The app is fully dockerized
- Designed to run locally only

# Installation
```bash
bash app/scripts/setup.sh
```

# How to Run
```bash
bash app/scripts/start.sh
```
